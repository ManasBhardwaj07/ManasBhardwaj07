## About Me

Backend-focused full stack developer building scalable real-time and asynchronous systems using Node.js, Redis, Docker, PostgreSQL, MongoDB, and AWS.

My primary focus is backend engineering, distributed systems, queue-driven architectures, real-time synchronization, and production-oriented full stack applications.

Currently building systems involving:

- WebSocket-based real-time collaboration
- RBAC & JWT authentication

- scalable API architectures

## Flagship Projects

### 🚀 CodeForge
Full-stack asynchronous code execution platform with isolated Docker sandbox execution, Redis + BullMQ worker architecture, deterministic verdict aggregation, and PostgreSQL-backed execution lifecycle tracking.

**Tech:** Next.js, TypeScript, PostgreSQL, Prisma, Redis, BullMQ, Docker

---

### ⚡ QueueForge
Distributed asynchronous job processing system designed around queue-driven execution, retry strategies, worker isolation, structured logging, and fault-tolerant processing.

**Tech:** Node.js, Redis, Docker, AWS EC2, BullMQ

---

### 🤝 ProjectHub
Real-time collaboration platform supporting concurrent project and task synchronization using Socket.IO, Redis session optimization, JWT authentication, and role-based access control.

**Tech:** MERN Stack, Socket.IO, Redis, AWS

---

### 🏌️ ApexDrive
Subscription-based golf rewards platform with Stripe subscriptions, Supabase RLS, webhook processing, admin analytics, and production-grade backend workflows.

**Tech:** React, Node.js, Supabase, PostgreSQL, Stripe, Vercel
